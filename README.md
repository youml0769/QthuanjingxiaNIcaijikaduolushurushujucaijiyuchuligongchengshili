# Qt环境下NI采集卡多路输入数据采集与处理工程实例

## 项目描述

本项目提供了一个在Qt环境下实现的NI采集卡多路输入电压的同时采集、显示、曲线绘制及文本存储的工程应用实例。该工程文件旨在帮助开发者快速上手NI采集卡的C++编程，并解决在实际应用中可能遇到的数据采集与处理问题。

## 功能特点

- **多路输入数据采集**：支持NI采集卡的多路输入电压数据的同时采集。
- **实时数据显示**：通过Qt界面实时显示采集到的数据。
- **曲线绘制**：使用开源库qcustomplot进行曲线绘制，直观展示数据变化趋势。
- **文本存储**：将采集到的数据保存为文本文件，便于后续分析和处理。
- **双线程操作**：采用双线程设计，确保数据采集与界面显示的流畅性。

## 使用说明

1. **环境配置**：
   - 确保已安装Qt开发环境。
      - 安装NI采集卡的驱动及开发库。
         - 下载并配置qcustomplot库。

         2. **编译与运行**：
            - 克隆或下载本仓库的工程文件。
               - 使用Qt Creator打开工程文件。
                  - 配置项目依赖项，确保所有库文件路径正确。
                     - 编译并运行项目。

                     3. **操作指南**：
                        - 启动程序后，界面将显示NI采集卡的实时数据。
                           - 可以通过界面控件调整采集参数，如采样率、通道选择等。
                              - 数据将实时显示在曲线图上，并自动保存为文本文件。

                              ## 依赖库

                              - **Qt**：用于界面开发和事件处理。
                              - **NI采集卡驱动**：用于数据采集。
                              - **qcustomplot**：用于数据曲线绘制。
